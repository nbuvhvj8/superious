# Future Feature Roadmap

Based on the current strategic goals of the outlier research engine, the following 5 features are proposed for future development:

1. **Interactive AI Script Refiner (Chat)**
   - A dedicated side-panel allowing users to iteratively refine generated scripts through natural language (e.g., "make it punchier", "reformat for a Gen-Z audience").

2. **Automated Trend Monitoring (Cron Job)**
   - Scheduled research jobs that re-scan the web for new data on specific topics, automatically notifying the user if a script needs updating due to new developments.

3. **Semantic Asset Library (Storages)**
   - A visual vault for all screenshots, PDF sources, and AI-generated B-roll prompts collected during research, tagged and searchable by topic.

4. **Multi-Platform Export Sync**
   - Direct integration to push research and script drafts to external platforms like Notion, Slack, or Trello for seamless agency collaboration.

5. **Collaborative Workspace & Annotations**
   - Real-time collaboration features allowing teams to leave comments, annotations, and feedback directly on research sources and script drafts.
